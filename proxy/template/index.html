openit.io

View hosted source code of packages. Usage : https://openit.io/<alias>/<package-name> Examples :

npm

Aliases: /npm, /js, /javascript

  • Normal package: https://openit.io/js/preact
  • Scoped package: https://openit.io/js/@phenomic/core

For power users

Add these function to your .bashrc/.profile:

Openit right from CLI

function openit() {
  open "http://openit.io/$1"
}

Then call it 

$ openit js/preact

Git clone with openit

function clone() {
  git clone $(curl -Ls -o /dev/null -w %{url_effective} https://openit.io/$1)
}

Then call it 

$ clone js/preact
